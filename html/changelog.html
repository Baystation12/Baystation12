03 August 2020

Albens updated:

  • Changes to the descriptions of the Supermatter Specialty Control buttons.
  • Adds additional secure area signs around atmos maintenance.

DeadmonWonderland updated:

  • Adds Old Lab 2, an exoplanet spawn.

Devildabeast updated:

  • The Union Enforcer and Union Representative now start with empty pockets.
  • The contractor Shuttle Pilot no longer spawns with a jacket.
  • The contractor Forensic Technician no longer spawns with a coat and hat.
  • The Paramedic alternate title no longer gets an EMT belt.

Kearel updated:

  • Tendrils now penetrate armor and deal more damage.
  • Biters are now a tier 1 item.
  • Shooters have more range and their projectiles weaken and embed. They are now a tier 2 item.
  • Doubled the number of resources nutrient siphons give.

Rowtree updated:

  • Adds more drinks to the lunchbox selection.
  • Adds Lemon Juice to the Booze-O-Mat
  • Coconut Milk no longer requires Nutriment.
  • Removes Brown Star, as it was redundant with the addition of Orange Cola
  • Replaces Starkist cans, which were Brown Star with Irn Bru.
  • Nevadan Gold Whiskey no longer requires Irish Coffee, instead uses Irish Cream as intended.
  • Tweaked Booze-O-Mat Layout and in some cases, added the alcohol type to the bottle name.
  • Lowered the amount of juice cartons to encourage botany play.

Shadowtail117 updated:

  • Admins can now specify a reason when dispatching an Emergency Response Team that will be shown to the team when they spawn in.
  • The Emergency Response Team channel has its own key now: "O"/"o".
  • The Emergency Response Team welcome text has been modified to be more clear on who the leader is and your role in the game. It also includes the reason specified by the admin that dispatched it.
  • Added an all-in-one telecomms machine to the ERT shuttle that allows them to speak on their own private frequency at the base, in transit, and on the Torch (or anywhere their shuttle is). It is not advised to mess with it even if you have the tools to do so.
  • Emergency Response Team headsets now have their default channel set to their private frequency. This can be manually changed back to the common frequency, but doing so is irreversible!
  • Added a reagent grinder to the medical section of the Emergency Response Team base.
  • The "SEV Torch" is now considered an improper name, meaning "the" will be added before it in some cases. The same goes for the FTV Bearcat.

SierraKomodo updated:

  • Crew Monitor now sorts properly by name instead of by deck.
  • Crew Monitor now reads off connected z_levels. Might affect how sensors work on off-ship z_levels.
  • Broken decals, turfs, wall-mounted objects, furniture, etc should no longer be broken.

TheLordME updated:

  • Meters can now be clicked directly to read the value, instead of haveing to examin them. Works only from adjacent tiles for Non-Cyborgs.
  • AIs can now read Meters by clicking on them.

30 July 2020

NullSnapshot updated:

  • Replaces current chat system with an HTML based one, allowing for a lot of more powerful functions including...
  • Dark mode options are now available for the game, which toggles the color of the chat bar verb controls, and game background.
  • Chat now features a ping counter allowing you to see your current quality of connection to the server
  • String highlighting! You now have the ability to highlight up to 5 different strings in chat to draw your attention.
  • New Message highlighting. While scrolling back through chat, you will now be informed of any new messages.
  • Line combining. Repeat messages sent one after another will combine in order to reduce clutter.
  • Independently configurable line and icon sizes, allowing you to independently change the size of each.
  • configurable line height, allowing you to change the distance between individuals lines in the chat box.
  • You can now change the default chat font to any font supported on your system.
  • Press alt-enter to toggle full screen
  • fit-viewport verb which automatically resizes the divider to exactly fit the game window.
  • Game window layout slightly changed to accommodate these new features, namely the input bar has been moved to directly underneath the chat window.

24 July 2020

NullSnapshot updated:

  • Replaces current chat system with an html based one called vchat, allowing for more powerful functions including...
  • Dark mode is now in the game, which toggles the color of the chat bar, stats/verb panel, and game background.
  • Chat now features a ping counter allowing you to see the current connection status with the server.
  • Chat tabs! The chat window now supports multiple tabs through its control menu, allowing you to filter chat on a per tab basis.
  • New message highlighting will indicate to you when new messages are visible to you across any tab
  • Line combining. Repeat messages sent one after another will combine in order to reduce clutter.
  • animated messages. User toggle setting to have text fade in instead of their normal behavior.
  • Configurable line height, allowing you to change the distance between lines in the chat box.
  • Chat persistence. If you lose connection, when you reconnect you'll automatically retain any of the last 200 messages you've seen.
  • Vchat preference toggle added. If you'd prefer to use the legacy old chat, you can do so by toggling Use Fancy Chat in your preferences. Changing between the two should be more or less seamless, as the last 200 messages will be preserved in either mode.
