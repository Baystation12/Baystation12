04 April 2020

  • Inflatables can no longer be deployed on any object that prevents a standard powered airlock to close
  • Unpowered doors can no longer close on any object that would prevent a standard powered airlock from closing.

  • Adherent with usable levitation plates can now feather-fall, avoiding damage
