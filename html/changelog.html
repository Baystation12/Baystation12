afterthought2 updated:

  • The shield generator has been reworked. It is now much more expensive to keep the shields on continuously, and for large shields it's likely not viable.
  • The shield generator now has an idle and active state (as well as the off state). In the idle state, it consumes limited power and does not generate a shield. In the active state, it consumes a lot of power and produces a shield.
  • If an idle generator is toggled on, it takes a short amount of time to spin up to active state. If an active generator is toggled to idle, it will switch to that state instantly.
  • There are several levels of idle power usage that can be selected. Higher power usage decreases the spin-up time proportionately.
  • The shield generator takes time to adjust the radius of the shield. The radius will not be adjusted while the shield is active.
  • The shield generator's power storage has been vastly expanded.
  • The shield generator's field integrity is now tracked separately from the power storage (it's determined by the amount of power stored, up to a threshold).

afterthought2 updated:

  • Machines now require power components to draw power. These can be printed from the autolathe. The tesla link one is the default one to use.
  • Many machines now require screens and keyboards to be fully interactable. You can print them from the autolathe.
  • To install components, open the panel with a screwdriver (after the machine is built) and click on the machine with the component. To remove components, use the wrench while the panel is open.
  • Computer construction has been changed. A screen and keyboard is required now. The order is: make a frame, wrench to anchor the frame, add 5 coils, add the circuit, add all components as prompted, screwdriver to build. To deconstruct, crowbar to remove components, wirecutters to cut the wire, unwrench, and apply welder.
  • General machinery follows the same construction steps as computers now.
  • Computers now have panels, like other machines. To deconstruct a computer, use a screwdriver to open the panel and then a crowbar to deconstruct.
  • Newly built machines have the panel open.
  • If confused about this, examine the machine while standing next to it to get information about which tools to use on it.
