code/modules/admin/connectioncheck/connectioncheck_functions.dm

/proc/_fetch_connections Checks for prior connections matching ckey, ip, or cid.
/proc/_unique_ckeys_from_connections Returns a sorted list containing only each unique ckey present in a list of connections provided by _fetch_connections().
/proc/_unique_cids_from_connections Returns a sorted list containing only each unique CID present in a list of connections provided by _fetch_connections().
/proc/_unique_ips_from_connections Returns a sorted list containing only each unique IP present in a list of connections provided by _fetch_connections().
/proc/_show_associated_connections Generates and displays an HTML window, displaying data from a _fetch_connections() call with the provided parameters.
