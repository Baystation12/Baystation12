code/modules/admin/connectioncheck/connectioncheck_functions.dm

/proc/_fetch_connections Checks for prior connections matching ckey, ip, or cid.
/proc/_fetch_bans Checks for bans matching ckey, ip, or cid.
/proc/_find_bans_in_connections Checks a list of connections for bans matching any of the list entries.
/proc/_unique_ckeys_from_connections Returns a list containing only each unique ckey present in a list of connections provided by _fetch_connections().
/proc/_unique_cids_from_connections Returns a list containing only each unique CID present in a list of connections provided by _fetch_connections().
/proc/_unique_ips_from_connections Returns a list containing only each unique IP present in a list of connections provided by _fetch_connections().
