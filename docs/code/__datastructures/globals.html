/var/GLOBManaged Globals datum. Any defined global vars can be accessed via GLOB.varname.
GLOBAL_INITInitializes the GLOB member to V.
GLOBAL_VAR_CONSTInitializes the global constant GLOB.[X] with the value [V] as a managed global.
GLOBAL_VARInitializes the global variable GLOB.[X] with no value (null) as a managed global.
GLOBAL_VAR_INITInitializes the global variable GLOB.[X] with the value [V] as a managed global.
GLOBAL_DATUMInitializes the global variable GLOB.[X] as an instance of /static[P] with no value (null) as a managed global.
GLOBAL_DATUM_INITInitializes the global variable GLOB.[X] as an instance of /static[P] with the value [V] as a managed global.
GLOBAL_LISTInitializes the global variable GLOB[X] as a list with no value (null) as a managed global.
GLOBAL_LIST_INITInitializes the global variable GLOB[X] as a list with value [V] as a managed global.
GLOBAL_LIST_EMPTYInitializes the global variable GLOB[X] as an empty list (list()) as a managed global.
GLOBAL_PROTECTPrevents the GLOB member from being shown in View Variables.
