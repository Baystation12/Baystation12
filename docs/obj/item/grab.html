proc
use_sanity_check
(/atom/target, flags)

Validates that assailant can still perform an action with affecting and target. Performs some grab-specific checks, then passes through to assailant.use_sanity_check() with both src and affecting.

Parameters:

  • target - The atom being interacted with.
  • flags (Bitflag, any of SANITY_CHECK_*, default SANITY_CHECK_DEFAULT) - Bitflags of additional settings. See code\__defines\misc.dm.

Returns boolean.
