Vars

ethernetHard-wired, therefore always on, ignores NTNet wireless checks.
identification_idIdentification ID. Technically MAC address of this device. Can't be changed by user.
identification_stringIdentification string, technically nickname seen in the network. Can be set by user.
long_rangeLong-range cards have stronger connections, letting them reach relays from connected Z-levels.
proxy_idIf set, uses the value to funnel connections through another network card.
