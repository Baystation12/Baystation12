setdirs
– /list

List ("number" -> instances of /datum/fluidtrack). Map of directional bit flags to attached fluidtrack isntances.

Indexes are stringified bitflags of the four main cardinal directions, duplicated once. The first set is incoming footsteps, and the second outgoing.

Quick reference of each bitflag:

INCOMING_NORTH = 1
INCOMING_SOUTH = 2
INCOMING_EAST = 4
INCOMING_WEST = 8
OUTGOING_NORTH = 16
OUTGOING_SOUTH = 32
OUTGOING_EAST = 64
OUTGOING_WEST = 128
